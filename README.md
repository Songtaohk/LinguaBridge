# LinguaBridge AI

LinguaBridge AI is a professional, high-performance web application designed to translate multilingual audio content into natural-sounding speech (Chinese or English) and provide smart summaries.

## ‚ú® Features

- **Multi-language Support**: Automatically detects and translates from 100+ languages.
- **Bi-directional Output**: Choose between Chinese and English as your target translation language.
- **Smart Link Resolver**: Snippets audio from blogs, news sites, or even article text to read it aloud.
- **AI Summary**: Generates key insights from any audio content using Gemini 3.
- **Voice Synthesis**: Uses high-quality neural voices for the best listening experience.
- **Bookmarklet Support**: One-click translation of external webpages.
